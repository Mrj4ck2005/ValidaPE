# Valida-PE 🇵🇪

**Verificación de Identidad y Datos Fiscales Peruanos**

Aplicación web moderna para consultar información de:
- **RENIEC**: Registro Nacional de Identidad (DNI)
- **SUNAT**: Superintendencia Nacional de Aduanas y de Administración Tributaria (RUC)

---

## 🚀 Características

✅ Consulta instantánea de DNI (RENIEC)
✅ Consulta instantánea de RUC (SUNAT)
✅ Interfaz moderna y responsiva
✅ Aplicación Web Progresiva (PWA) - Instalable en dispositivos móviles
✅ Historial local de últimas 5 consultas
✅ Exportación a PDF de resultados
✅ Arquitectura Cliente-Servidor segura
✅ Rate limiting y protección contra abuso

---

## 📋 Requisitos Previos

- Node.js 16+
- npm o yarn
- Cuenta en Decolecta con token API válido
